# HR Analytics Dashboard

## Project Overview
This project analyzes employee attrition, salary, job satisfaction, and workforce distribution using Power BI.

## Tools Used
- Power BI
- SQL
- Excel

## KPIs
- Total Employees
- Employees Left
- Attrition Rate
- Average Salary Hike
- Average Job Satisfaction

## Dashboard Features
- KPI Cards
- Department Filter
- Gender Filter
- Job Role Filter
- Years at Company Filter
- Attrition Filter
- Department-wise Attrition
- Gender Distribution
- Average Salary by Job Role
- Performance vs Job Satisfaction
- Department-wise Employee Distribution
